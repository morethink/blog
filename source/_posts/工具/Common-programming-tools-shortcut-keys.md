#  Navicat for mysql 快捷键


1. Ctrl+Q           打开查询窗口
2. Ctrl+/            注释sql语句
3. Ctrl+Shift +/  解除注释
4. Ctrl+R           运行查询窗口的sql语句
5. Ctrl+Shift+R   只运行选中的sql语句
6. F6               打开一个mysql命令行窗口
7. Ctrl+L            删除一行
8. Ctrl+N           打开一个新的查询窗口
9. Ctrl+W          关闭一个查询窗口
